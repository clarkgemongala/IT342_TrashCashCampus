# 🗑️ TrashCashCampus

## Overview
TrashCashCampus is a waste management system that promotes recycling on campus through education and rewards. The platform includes both a web and mobile app for tracking recycling habits, QR code waste verification, and a point-based reward system. Users can redeem points for perks, while analytics provide insights into recycling performance. The system also features smart bin monitoring and maintenance alerts. Designed for students, staff, and administrators, TrashCashCampus enhances sustainability efforts, improves waste tracking, and fosters responsible disposal practices for a greener campus.

## Features

### 🔑 User Authentication and Management
- Single sign-on integration with university credentials

### 🗑️ Waste Validation and Tracking
- Real-time image recognition for waste type verification
- Uses phone camera to scan QR codes on recycling bins
- Tracks individual recycling history
- Awards points instantly for proper disposal

### 🎁 Points and Rewards System
- Browse and redeem available rewards
- Notify users of points updates after successful recycling

### 📊 Analytics and Reporting
- Visual representation of personal recycling statistics

### 🏗️ Bin Management System
- Smart bin monitoring and maintenance alerts
- Unique QR codes on bins provide specific recycling tips

### 🌱 Community Engagement Features
- Environmental impact visualization
- Educational feedback upon correct waste disposal

### 📱 Minimal Navigation
- 3-4 main menu items: Home, My Points, Find a Bin, Rewards
