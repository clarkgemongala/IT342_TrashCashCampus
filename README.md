## 📋 Overview
TrashCash Campus is an innovative recycling platform designed to promote sustainable waste management practices on university campuses. The system incentivizes students and staff to recycle by providing a gamified experience through QR code scanning, waste validation, and a rewards system.

## ✨ Core Features

### 1. 🗑️ Smart Recycling System
- **🔍 QR Code Bin Scanning**: Users can scan QR codes on recycling bins using their mobile devices
- **📸 Waste Validation**: Real-time image recognition to verify correct waste disposal
- **⭐ Points System**: Earn points for proper recycling practices
- **🎁 Rewards Marketplace**: Redeem points for various rewards and incentives

### 2. 👤 User Experience
- **🔐 Single Sign-On**: Seamless integration with university credentials
- **📱 Mobile-First Approach**: Native Android application for on-the-go recycling
- **💻 Web Dashboard**: Comprehensive web interface for account management and analytics
- **📊 Real-time Tracking**: Monitor recycling activities and points in real-time

### 3. 👨‍💼 Administrative Features
- **🗑️ Bin Management**: Track and manage recycling bins across campus
- **📈 Analytics Dashboard**: Monitor recycling patterns and user engagement
- **🎯 Rewards Management**: Create and manage the rewards catalog
- **👥 User Management**: Administer user accounts and permissions

## 🚀 Getting Started

### Prerequisites
- Node.js (for web development)
- Android Studio (for mobile development)
- Firebase account
- University email domain access

### Installation

#### Web Application
```bash
cd frontend_web/trashcash-campus
npm install
npm run dev
```

#### Mobile Application
1. Open `frontend_mobile/TrashCashCampus-mobile` in Android Studio
2. Configure Firebase settings
3. Build and run on an Android device or emulator

### Configuration
1. Set up Firebase project
2. Configure authentication providers
3. Initialize Firestore database
4. Set up storage buckets
5. Configure CORS settings
