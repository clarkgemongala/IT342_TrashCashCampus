# IT342_TrashCashCampus  

## Overview  
TrashCampus is a waste management system that promotes recycling on campus through education and rewards. It includes a web and mobile app for tracking recycling habits, QR code waste verification, and a point-based reward system. Users can redeem points for perks, while analytics provide insights into recycling performance. The system also features smart bin monitoring and maintenance alerts. Designed for students, staff, and administrators, TrashCampus enhances sustainability efforts, improves waste tracking, and fosters responsible disposal practices for a greener campus.  

## Feature List  

### User Authentication and Management  
- Single sign-on integration with university credentials  

### Waste Validation and Tracking  
- Real-time image recognition for waste type verification  

### Points and Rewards System  
- Browse and redeem available rewards  


### Analytics and Reporting  
- Visual representation of personal recycling statistics  

### Bin Management System  
- Uses phone camera to scan QR codes on recycling bins, Verifies correct waste type through image recognition,   Tracks individual recycling history, Awards points instantly for proper disposal 

 

### Community Engagement Features  
- Environmental impact visualization  

### Bin-Specific Recycling Tips  
- Each bin's QR code contains unique tips for proper usage  

### Educational Feedback  
- App confirms correct waste disposal and displays a brief fact  

### Notifications and Alerts  
- Notify users of points updates after successful recycling  

### Minimal Navigation  
- 3-4 main menu items: Home, My Points, Find a Bin, Rewards
